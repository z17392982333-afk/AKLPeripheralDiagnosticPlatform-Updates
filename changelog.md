# Changelog

## v1.8.62-mouse-standard-judgement-ui - 2026-06-11

- 鼠标标准化流程增加总状态、当前项目、项目结果列表和异常明细展示。
- 标准流程扩展为 9 项：左键、右键、中键、侧键 1、侧键 2、滚轮上滚、滚轮下滚、移动、左键拖动。
- 加入自动异常判定：疑似双击/抖动、重复按下未释放、卡键/释放延迟、非当前项目误触、滚轮反向、移动轨迹突变、拖动中断。
- 每项完成弹窗会显示自动检测结论；如果自动检测发现异常，即使人工确认正常也会落为观察，避免异常被直接覆盖。
- 中止检测会同步停止标准流程、鼠标检测会话和 UI 状态。
## v1.8.62-mouse-standard-flow - 2026-06-11

- 新增鼠标标准化检测流程入口。
- 左键、右键、中键、侧键 1、侧键 2、滚轮上滚、滚轮下滚均按 100 次/格作为一轮标准项。
- 每项达标后弹出确认窗口，可选择正常、观察、异常或重测。
- 标准化流程运行时暂停 RawInput 异常复测，并阻止自由独立检测重复记录。
- 流程 UI 做节流更新，减少低配置门店电脑上的刷新压力。

## v1.8.62-wheel-delta-notch-fix - 2026-06-10

- 修复鼠标滚轮回滚检测口径。
- 将滚轮事件从“事件次数”改为“标准滚轮格数”累计。
- 支持小于 120 的 WheelDelta 累积，避免高精度滚轮小 Delta 被丢弃。
- 回滚判定按“稳定同向格数 + 反向格数 + 快速回到原方向”计算。
- 主窗口内嵌鼠标测试和独立鼠标检测窗口使用同一路径。
