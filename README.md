# AKL Peripheral Diagnostic Platform Updates

This repository is the public update metadata channel for AKL Peripheral Diagnostic Platform.

It does not store source code. Source code is maintained separately in:

https://github.com/z17392982333-afk/AKLPeripheralDiagnosticPlatform

## Files

- `latest.json`: current release metadata used by the application update checker.
- `changelog.md`: human-readable release history.
- `releases/<version>/notes.md`: release notes for each build.

Installer binaries should be uploaded as GitHub Release assets, not committed to Git.
